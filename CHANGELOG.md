# Change Log

All notable changes to the "ask-project-manage" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [26.603.1] - 2026-06-03

- 更新扩展版本规则为年月日次数格式，当前版本为 26.603.1
- 优化扩展市场展示信息，补充关键词和分类，提升 VS Code / Open VSX 检索匹配度
- 新增首次使用引导，说明导入项目、底部状态栏入口和偏好设置入口
- 修复新手引导在面板关闭、重新激活或多窗口场景下重复弹出的问题
- 优化首页布局，移除底部圆环装饰，并将 HUD 统计面板贴近底部展示
- 修复顶部导航换行后遮挡第一排项目卡片的问题
- 优化列表滚动区域，滚动条靠右显示，并仅在内容确实需要滚动时出现
- 更新 README 为产品介绍和快速开始结构，降低新用户上手成本

## [1.0.0] - 2026-05-28

- 发布灵枢控制台视觉风格的项目管理面板
- 支持导入文件夹、导入工作区、项目分组和搜索
- 支持 JSON / YML 配置导入导出，用于备份和迁移
- 支持批量管理、删除确认、空状态引导和底部 HUD 统计
- 更新插件 Logo、页面 favicon 和社区展示文档
